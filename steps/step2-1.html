판매처 등록
풀필먼트 연동을 위해 품고 나우에 판매처 등록을 진행합니다.
진행 방법
① 품고 나우 > 주문 > 판매처 계정 > 판매처 계정 개별 등록을 클릭해주세요.
② 다음과 같이 설정 후 등록합니다.
- 판매처명 : 스마트스토어(물류사)
- 브랜드명 : 조회되는 브랜드명 선택
- 연동 계정 : 스마트스토어 API 연동용 판매자 ID 입력
확인 경로 : 스마트스토어 > 스토어 관리 > 서비스 연결 > API 관리 > API 연동용 판매자 ID
오른쪽 화면은 > 버튼으로 넘기면서 과정을 확인할 수 있어요.
